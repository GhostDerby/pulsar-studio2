# EXECUTION PROTOCOL – TANGO MODE

---

## Roles

Architect: Human (Project Owner)

System Brain: ChatGPT
- Architecture
- Decomposition
- Risk analysis
- Economic validation

Execution Engine: Cursor
- Code writing
- Refactoring
- Implementation
- Audit

---

## Workflow

1. Task defined by Architect.
2. Architecture defined via ChatGPT.
3. Execution done in Cursor.
4. Result validated against CORE_CONSTITUTION.
5. Iterate until stable.

---

## Restrictions

Cursor must NOT:
- Redesign architecture
- Introduce new patterns
- Change system structure
- Add features outside roadmap

ChatGPT must:
- Validate logic
- Protect architecture
- Enforce economic filter

---

## Priority Order

1. Revenue
2. Stability
3. Optimization
4. Expansion

No deviation allowed before 300€ milestone.

---

End of Protocol# EXECUTION PROTOCOL – TANGO MODE
