# ROADMAP TO 300€ MONTHLY PROFIT

Objective: Reach minimum 300€ stable monthly net profit.

---

## PHASE 1 – SYSTEM STABILIZATION (Week 1)

- [ ] Stabilize Linux environment
- [ ] Fix keyring and login issues
- [ ] Verify SSD health
- [ ] Confirm GitHub backup operational
- [ ] Validate local development workflow
- [ ] Confirm Cursor + ChatGPT execution protocol working

Deliverable: Stable working development environment.

---

## PHASE 2 – PRODUCT FINALIZATION (Week 1–2)

Target Product: 30s AI Product Video (Brazil Market)

- [ ] Finalize 9:16 Brazil vertical template
- [ ] Create Smartwatch demo (Amazon BR)
- [ ] Create Smartwatch demo (Mercado Livre)
- [ ] Standardize production pipeline
- [ ] Lock production settings (resolution, pacing, color, subtitles)
- [ ] Calculate exact unit production cost

Deliverable: 1 replicable production template.

---

## PHASE 3 – OFFER DEFINITION

- [ ] Define entry product (clear scope)
- [ ] Define fixed entry price
- [ ] Define delivery time
- [ ] Define cost structure
- [ ] Define minimum margin target

Deliverable: Sellable fixed offer.

---

## PHASE 4 – FIRST SALES EXECUTION

- [ ] Identify 10 active sellers
- [ ] Create watermarked demo
- [ ] Send outreach message
- [ ] Close first 3–5 paid orders

Target: 3–5 clients × entry offer = 300€+

---

## RULE

No super-features before stable revenue.

Revenue first. Expansion later.
